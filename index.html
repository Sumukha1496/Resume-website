PROJECTS

  • SMART VAULT :

    An automated Bank Locker system which provides security by eliminating human entry in the Locker Room.

    Description

    The present bank locker systems involves two keys. One Master-key is with the bank manager and another key is with the user.So it involves human entry into the locker room. Moreover we can not get 24 x 7 service.

    So ICICI bank came up with an idea i.e, If we can have a robot which moves inside the sealed locker room and brings the locker box of the respective user we can eliminate the human entry into the locker room and can provide service for 24 x 7 like the ATM's.

    We started working on this project as a part of academics and came up with the working model of the above. We used

    • RFID cards to access different locker boxes of different users.
    • Arduino MicroController to control all the operations
    • Robotic arms and wheels for picking and moving
